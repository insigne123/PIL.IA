# **App Name**: BidWise

## Core Features:

- Dashboard Overview: Display key performance indicators (KPIs), recent tenders, and overall compliance status at a glance.
- Tender Management: Allow users to create, track, and manage tenders, including details like client, due date, status, and associated documents.
- Risk Assessment: Identify and assess potential risks associated with each tender, focusing on deadlines, compliance, and contractual obligations.
- Compliance Tracking: Monitor and track compliance progress for each tender, highlighting areas that require attention and improvement. Display as percentage for each section
- AI Copilot: Suggest the next best action by evaluating the stage of a bid, using a tool to assess available resources and deadlines, and comparing them against the likelihood of winning. Use this analysis to show users a list of automatable processes such as matrix generation and task creation,.
- Document Upload and Analysis: Enable users to upload tender documents for AI-powered analysis, including matrix generation, risk assessment, and task identification.
- Task Management: Create and assign tasks related to each tender, set deadlines, and track progress to ensure timely completion of deliverables.

## Style Guidelines:

- Primary color: Saturated indigo (#6366F1) to convey intelligence, modernity, and trustworthiness.
- Background color: Very light, desaturated indigo (#F0F1FA) for a clean, professional backdrop.
- Accent color: Vivid violet (#A855F7) to highlight key actions and elements.
- Body and headline font: 'Inter' (sans-serif) for a modern, clean, and easily readable interface.
- Use a consistent set of line icons from Lucide to ensure clarity and visual appeal.
- Implement a structured layout with clear sections and consistent spacing to improve usability.
- Employ subtle transitions and animations to enhance the user experience and provide visual feedback.
