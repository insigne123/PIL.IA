# Firebase Studio

This is a NextJS starter in Firebase Studio.

To get started, take a look at src/app/page.tsx.

## Regression Testing (CAD Extraction)

This project includes a regression runner to valid quantity extraction from DXF files.

See [docs/regression.md](docs/regression.md) for detailed instructions.

Quick start:
```bash
npm install
git lfs pull
npm run regression
```
